# SharpC2
.NET C2 Framework Proof of Concept

## Features
* Modular design
* Supports fork-and-run operations
* PPID Spoofing
* Process Mitigation Policy (Block DLLs)
* ETW Patching
* Reverse Port Forwarding
* API control with Swagger UI
